# 智能表单

AI Module的核心功能之一是智能表单操作，让AI能够理解和操作Entity Engine的表单组件。

## 基本使用

### 启动AI对话

1. 打开任何Entity Engine表单页面
2. 在表单右上角找到"智能填表"按钮
3. 点击按钮打开AI对话界面
4. 开始与AI对话

### 简单表单填写

AI可以理解自然语言指令并自动填写表单：

```
用户：帮我填写用户信息，姓名是张三，年龄25岁，邮箱是zhangsan@example.com

AI：好的，我来帮您填写用户信息。
[AI自动填写对应字段]
✅ 姓名：张三
✅ 年龄：25
✅ 邮箱：zhangsan@example.com
表单信息已填写完成。
```

## AI能执行的表单操作

### 获取表单信息

**查看所有字段**
```
用户：这个表单有哪些字段？

AI：当前表单包含以下字段：
- 姓名 (name) - 文本类型，必填
- 年龄 (age) - 数字类型，可选
- 邮箱 (email) - 邮箱类型，必填
- 电话 (phone) - 文本类型，可选
```

**查看当前值**
```
用户：表单现在填写了什么内容？

AI：当前表单的填写内容：
- 姓名：张三
- 年龄：25
- 邮箱：zhangsan@example.com
- 电话：（未填写）
```

### 智能字段填写

**批量填写**
```
用户：帮我创建一个管理员账户，名字叫李四，邮箱用公司邮箱

AI：我来帮您创建管理员账户。
✅ 姓名：李四
✅ 角色：管理员
✅ 邮箱：lisi@company.com
✅ 状态：激活
管理员账户信息已填写完成。
```

**智能推断**
```
用户：这是一个客服人员

AI：了解，我来填写客服人员信息。
✅ 角色：客服
✅ 部门：客户服务部
✅ 工作时间：9:00-18:00
✅ 状态：在职
客服人员信息已设置完成。
```

### 表单验证

**检查表单是否正确**
```
用户：检查一下表单填写是否正确

AI：让我检查表单填写情况：
✅ 姓名：已填写，格式正确
✅ 邮箱：已填写，格式正确
❌ 电话：格式不正确，应为11位数字
⚠️  年龄：建议填写，虽然不是必填项

发现1个错误，建议修正后再提交。
```

### 表单重置

**清空重新开始**
```
用户：重新开始，清空所有内容

AI：好的，我来重置表单到初始状态。
✅ 表单已重置，所有字段已清空
您可以重新开始填写信息。
```

## 高级使用技巧

### 上下文理解

AI具有上下文记忆能力，可以进行多轮对话：

```
用户：填写一个新员工信息
AI：好的，请告诉我新员工的基本信息。

用户：姓名王五，25岁
AI：✅ 姓名：王五  ✅ 年龄：25
还需要其他信息吗？

用户：他是开发工程师
AI：✅ 职位：开发工程师  ✅ 部门：技术部
已根据职位自动填写相关信息。

用户：邮箱用他的名字加公司域名
AI：✅ 邮箱：wangwu@company.com
新员工信息填写完成。
```

### 条件逻辑处理

AI可以理解字段间的关联关系：

```
用户：如果是VIP客户，给他设置最高等级

AI：了解，我来设置VIP客户信息。
✅ 客户类型：VIP
✅ 等级：钻石级
✅ 折扣：9.5折
✅ 专属服务：开启
VIP客户设置已完成。
```

### 数据格式化

AI会自动处理数据格式：

```
用户：电话号码是一三八一二三四五六七八

AI：✅ 电话：13812345678
已自动格式化电话号码。
```

## 支持的字段类型

AI Module支持Entity Engine的所有标准字段类型：

| 字段类型 | AI能力 | 示例 |
|----------|--------|------|
| 文本 | 智能填写、格式验证 | 姓名、地址、备注 |
| 数字 | 数值计算、范围验证 | 年龄、价格、数量 |
| 邮箱 | 格式验证、自动生成 | 工作邮箱、个人邮箱 |
| 电话 | 格式化、验证 | 手机、座机 |
| 日期 | 日期解析、计算 | 生日、入职时间 |
| 选择 | 智能匹配选项 | 性别、状态、类型 |
| 布尔 | 逻辑判断 | 是否激活、是否VIP |

## 最佳实践

### 清晰的指令

**推荐的表达方式：**
```
✅ "帮我填写用户张三的基本信息，年龄30岁"
✅ "检查表单是否填写正确"
✅ "将状态设置为激活"
```

**避免的表达方式：**
```
❌ "随便填点什么"
❌ "你看着办"
❌ "填个差不多的"
```

### 分步骤操作

对于复杂表单，建议分步骤进行：

```
1. 先填写基本信息
2. 再设置权限和角色
3. 最后检查和验证
```

### 充分利用AI的智能性

- 让AI根据上下文自动推断信息
- 使用自然语言描述需求
- 利用AI的验证和建议功能

## 故障排除

如果AI无法正确操作表单，请检查：

1. **字段权限** - 确认字段是可编辑的
2. **字段类型** - 确认数据类型匹配
3. **必填验证** - 确认必填字段已填写
4. **格式要求** - 确认数据格式符合要求

更多问题解决方案请查看[常见问题](./troubleshooting)页面。
