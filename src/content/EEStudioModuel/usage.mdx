# 使用教程

通过可视化界面编辑 Entity Engine 的数据模型和视图配置。

## 打开 Studio

点击 Entity Engine 应用右上角的设置菜单，选择**"打开 Studio"**。

## 模型编辑

### 创建新模型

1. 点击模型选择器旁的**"新建模型"**按钮
2. 输入模型名称（如：`Product`）
3. 添加模型描述（可选）

### 添加字段

1. 点击**"添加字段"**按钮
2. 设置字段信息：
   - **字段名**: 英文标识符（如：`title`）
   - **显示名**: 中文标签（如：`产品标题`）
   - **字段类型**: 选择数据类型
   - **是否必填**: 开启/关闭必填验证

### 常用字段类型

- **Text**: 单行文本输入
- **Textarea**: 多行文本输入  
- **Number**: 数字输入
- **Select**: 下拉选择
- **Boolean**: 开关控件
- **Date**: 日期选择器

### 字段排序

拖拽字段项可以调整字段在表单中的显示顺序。

## 视图编辑

### 选择视图类型

Studio 支持三种视图类型：
- **表单视图**: 用于数据录入和编辑
- **列表视图**: 用于数据列表展示
- **详情视图**: 用于数据详情展示

### 配置字段显示

1. 从左侧模型字段列表选择要显示的字段
2. 拖拽到右侧视图配置区域
3. 设置字段在视图中的显示标签
4. 调整字段显示顺序

### 字段选项设置

对于每个视图字段，可以设置：
- **显示标签**: 字段在界面上的显示名称
- **占用宽度**: 字段在表单中的宽度比例
- **是否只读**: 设置字段为只读状态
- **默认隐藏**: 设置字段默认不显示

## 实时预览

Studio 提供实时预览功能：

1. 点击**"预览"**按钮打开预览面板
2. 查看当前配置的实际效果
3. 支持不同设备尺寸预览

## 保存配置

### 保存更改

1. 完成编辑后，点击**"保存"**按钮
2. 确认保存对话框中的更改内容
3. 点击**"确认保存"**应用配置

### 取消更改

点击**"取消"**按钮可以放弃当前的所有更改。

## 操作技巧

- **拖拽排序**: 所有字段列表都支持拖拽排序
- **快速复制**: 右键字段可以快速复制字段配置
- **批量操作**: 使用 Ctrl/Cmd + 点击可以多选字段
- **实时保存**: 配置会自动检测变更并提示保存

## 常见配置示例

### 产品模型示例

```
模型名称: Product
字段配置:
- title (Text, 必填) - 产品标题
- description (Textarea) - 产品描述  
- price (Number, 必填) - 价格
- category (Select) - 分类
- inStock (Boolean) - 是否有库存
- createdAt (Date) - 创建时间
```

### 用户模型示例

```
模型名称: User
字段配置:
- username (Text, 必填) - 用户名
- email (Email, 必填) - 邮箱
- avatar (Image) - 头像
- role (Select) - 角色
- active (Boolean) - 是否激活
```
