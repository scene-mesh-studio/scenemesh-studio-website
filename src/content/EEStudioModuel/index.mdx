# Studio Module 概览

Studio Module 是 Entity Engine 的可视化配置工作室，让你通过拖拽界面编辑数据模型和视图，无需编写代码。

## 主要功能

- **可视化模型编辑**: 拖拽创建数据模型，设置字段类型和属性
- **视图配置**: 可视化设计表单、列表和详情页布局  
- **实时预览**: 所见即所得的配置效果预览
- **弹框式工作室**: 直接在应用中打开，无需页面跳转

## 快速开始

### 1. 打开 Studio

在 Entity Engine 应用中，点击右上角设置菜单，选择**"打开 Studio"**。

### 2. 选择或创建模型

- 从下拉菜单选择现有模型
- 或点击"新建模型"创建

### 3. 编辑模型

- 点击"添加字段"按钮
- 设置字段名称、类型和属性
- 拖拽调整字段顺序

### 4. 配置视图

- 在视图编辑器中选择要显示的字段
- 拖拽调整字段显示顺序
- 设置字段标签和样式

### 5. 预览和保存

- 使用实时预览查看效果
- 点击"保存"应用配置

## 支持的字段类型

- **文本**: 单行文本、多行文本
- **数字**: 整数、小数
- **日期**: 日期、时间、日期时间
- **选择**: 下拉选择、单选、多选
- **布尔**: 开关、复选框
- **关联**: 引用其他模型
