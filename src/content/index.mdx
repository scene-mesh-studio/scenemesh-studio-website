# SceneMesh Studio 文档中心

欢迎来到 SceneMesh Studio 文档中心。SceneMesh Studio 是一个专业的开发工作室，致力于提供创新的软件开发解决方案，帮助开发者构建现代软件应用。

## 📖 文档概览

本文档中心包含了 SceneMesh Studio 生态系统中所有产品的完整文档，包括快速入门指南、详细的API参考、最佳实践和实际应用案例。

## 🏗️ 产品架构

SceneMesh Studio 采用模块化架构设计，由两个核心产品组成：

### Stremind
**事件驱动的开发平台**，专为构建可扩展、智能的应用程序而设计。

**核心特性：**
- 实时事件处理引擎
- 原生AI模块集成
- 云原生可扩展架构
- 直观的开发者API和SDK

**适用场景：**
- 实时数据处理应用
- 事件驱动的微服务架构
- IoT和智能设备管理
- 需要AI集成的业务系统

### Entity Engine
**高级数据建模引擎**，提供类型安全的数据建模和智能模式生成功能。

**核心特性：**
- AI驱动的智能数据建模
- 类型安全的高级查询引擎
- 实时数据同步
- 企业级扩展性和性能

**适用场景：**
- 复杂数据结构建模
- 企业级业务应用开发
- 数据密集型应用
- 需要严格类型安全的项目
